# University Management System

This project is a **University Management System** developed as part of a database course project. It is designed to manage and organize university-related data efficiently.

## 📌 Features

* Student Information Management
* Teacher Information Management
* Course Management
* Enrollment System
* Database storage and retrieval

## 🚀 How to Run

1. Clone the repository
2. Open the project in your IDE
3. Setup the database
4. Run the main program

## 🎯 Project Goal

The goal of this project is to understand how database systems work and how to integrate them into real-world applications.
